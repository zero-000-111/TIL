# 11720 문제
## 풀이
```
n=int(input())
num = int(input())
sum=0
for i in reversed(range(0,n)):
    val = num//(10**i)
    sum+=val
    num -= val*(10**i)
print(sum)
```
## 코드 설명
- list(str(num))와 같이 리스트 형태로 변환하지 않는 제약을 전제로 작성
- 가장 왼쪽의 숫자부터 차례로 수를 도출하기 위해 몫을 연산하는 절차를 수행하고 수행이후의 수는 제거함으로써 다음 연산에 중복되지 않도록 주의함
- 문제점: 코드가 복잡해짐
## 대안 코드
```
n=int(input())
num=input()
sum=0
for i in num:
    sum+=int(i)
```
## 부가 설명
- 코드가 단순해짐과 동시에 여전히 리스트 형태로 변화하지 않아도 됨
